# Django Dictionary API

REST API для управления справочниками (словарями) на базе Django и Django REST Framework.

## Описание

Проект предоставляет API для работы со справочниками и их версиями. Поддерживает версионирование справочников с указанием дат актуальности.

## Основные возможности

- 📚 Управление справочниками (создание, чтение, обновление, удаление)
- 📅 Версионирование справочников с датами актуальности
- 🔍 Получение элементов справочника на конкретную дату
- 📖 Автоматическая генерация API документации (Swagger/OpenAPI)

## Модели данных

- **Dictionary** - справочник (имеет уникальный код и название)
- **DictionaryVersion** - версия справочника (содержит версию и дату начала)
- **DictionaryItem** - элемент справочника (связан с версией)

## Установка

```bash
# Установить зависимости
pip install -r requirements.txt

# Выполнить миграции
python manage.py migrate

# Запустить сервер
python manage.py runserver
```

## Тестирование

```bash
# Запустить все тесты
pytest -vv

```

## API Документация

Документация API доступна по адресу `/api/dosc` (Swagger UI).

## Стек технологий

- Django 6.0.4
- Django REST Framework 3.17.1
- drf-spectacular (для генерации OpenAPI документации)
- pytest & pytest-django (для тестирования)
